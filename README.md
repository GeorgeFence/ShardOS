ShardOS is Operating System based on Cosmos, created with C# by GeorgeFence

> [!WARNING]
> OS in progress

## RegistryManager
ShardOS includes registry to set up system settings or user settings:
- **Boot Options**: Boot to safe mode, No FS mode, Normal mode
- **Settings for UAS**
- **Appearance settings** - GUI, Desktop, Taskbar

## UAS
User Account System
- **Create User**
- **Delete User**
- **Set User active**
